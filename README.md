# PigTV
猪猪直播

- 一款简易的直播app,涉及:
- 如需正常运行播放,需手动添加ijkplayer静态库,
- 百度盘网址:[https://pan.baidu.com/s/1pKGmk59](https://pan.baidu.com/s/1pKGmk59) 密码: nwuv

## Support
- 1). Swift3.1版本
- 2). ysocket搭建服务器
- 3). ijkPlayer实现直播的拉流播放
- 4). LFLivekit实现视频的采集
- 5). 部分功能代码用RxSwift实现
- 6). 实现了粒子动画,礼物动画
